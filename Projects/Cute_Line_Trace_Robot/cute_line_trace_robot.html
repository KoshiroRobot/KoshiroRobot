Cute Line Trace Robot

About

From August to October 2023, I developed an educational line trace robot designed to teach both programming and electronics. This educational tool aims to provide an experience where learners can feel like the robot is their learning partner, combining adorable design with practicality. The robot features a board on its back for creating electronic circuits, allowing learners to gain knowledge of electronics in addition to programming. This project was carried out by a team of three members, with the technical aspects such as robot design being primarily my responsibility.

How it works

Miniaturization and Design Innovations

The robot is compact, with a height of about 13 cm. To efficiently use internal space, gears were used to offset the wheel rotation axes. To counter the bulky appearance caused by the battery placement, a design with a protruding structure at the back was adopted, balancing the overall look.

Electronics and Programming

The robot is equipped with an electronics board at the back, enabling users to engage in both programming and electronic assembly. This makes it a versatile educational tool for learning both hardware and software concepts.

Specs and Results

Features

  • Educational tool combining programming and electronics learning.
  • Compact and well-designed robot suitable for learning purposes.
  • Equipped with an electronics board for hands-on electronic assembly.

Use Cases

  • Tokuyama College Festival "Line Trace Project": Conducted two-day programming lessons during Tokuyama College's open campus. 15 children assembled the robots and successfully completed line tracing.
  • Programming Classes at After-School Programs in Nara Prefecture: Held six programming classes using this educational tool. The children learned programming concepts and eventually achieved line tracing with their own programs.
